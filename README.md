## How to Run:
 


#### Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`CLOUDINARY_CLOUD_NAME`
`CLOUDINARY_KEY`
`CLOUDINARY_SECRET`
`MAPBOX_TOKEN`

#### Install dependencies

```bash
  npm install
```

#### Start the server

```bash
  npm start
```
